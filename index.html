MINIMAL LIGHTBOX RESPONSIVE

SOURCE CODE DOWNLOAD

Initialize with HTML

Include the JavaScript at the bottom of site before the closing </body> 
<script src="minimal.lightbox.js"></script>
Add minimal-lightbox attribute in <img> tag. For example: 
<img minimal-lightbox src="image.jpg"/>

Thumbnail Option

Include in minimal-lightbox the original image, and in src the thumbnail image. For example: 
<img minimal-lightbox="image.jpg" src="image-thumbnail.jpg"/>
